解构赋值

过程是把线性结构的元素解开，并顺序的赋给其它变量，左边接纳的变量数要和右边解开的元素个数一致。

  • 代码示例
  • 这个过程就叫序列解包，或者递归解包，将多个值的序列解开，然后放到变量序列中。
  • 可以利用这个特性来交换两个变量的值
  • 假如需要获取（并且删除）字典中的 键-值 对时候，可以使用字典的 popitem 方法，这个方法可以返回 键-值。
  • 取得环境变量中的 键-值 对

通配解构

Python支持一个新特性，可以使用号运算符匹配多个元素来赋值，例如：a,b,\c = [1,2,3,4]，那么 c 的值是 [3,4]

需要注意的是，使用*号匹配到的变量总会放到一个列表中。

丢弃变量

在CPython中，变量名 _ 是合法的，它是个特殊的变量，保存着上一条命令的执行结果。

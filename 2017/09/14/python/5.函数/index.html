Python 函数

基本概念

  1. 函数是Python为了代码最大程度的重用和最小化代码冗余而提供的基本程序的结构
  2. 函数是将相关功能打包，并且参数化处理的可重复调用的代码段

函数分类

  1. 全局函数：定义在模块中
  2. 局部函数：嵌套于其他函数中
  3. lambda：匿名函数表达式
  4. 方法：与特定数据类型关联的函数，并且只能与数据类型关联一起使用

定义与销毁

定义

  1. 函数以 def 关键字开头，后面使用一个标识符作为函数的名称，以及一个括号来接收参数。
  2. 任何需要接收的参数必须定义在小括号内。
  3. 函数以 : 开始，为一个缩进的上下文。
  4. 函数的一行语句可以选择性的使用 """README""" 来存放一段函数的说明，将被显示在帮助内。
  5. 函数内使用 return [expression] 结束函数，返回一个或多个值，当返回多个结果时候会组合为一个元组对象，return 缺省会返回None。
  6. 函数使用 function_Name() 调用。
  • 示例

销毁

总体来说，函数在没有引用计数的情况下被销毁，影响引用计数的情况有：

  1. 新定义了重名的函数，原函数的引用计数会减1
  2. 使用 del 语句删除函数对象的时候，引用计数会减1

另外，在线程退出的时候，整个程序内的函数将被销毁

  • 引用计数的增加与删除

参数接收

Python支持函数支持：位置参数、默认值参数、可变位置参数、可变关键字参数

定义参数

默认情况下，函数对接受到的参数进行 从左向右 依此按照位置匹配得到参数。

有默认值的参数

在定义函数参数时候，为其设置一个默认值，那么当函数没有收到这个参数时，则使用其默认值。

  • 与位置参数混合使用时候

如果定义函数时混用位置参数和有默认值的参数，则有默认值的参数必须在位置参数之后。

可变数量位置参数（收集为元组）

定义函数时使用 * 开头的参数，可收集任意多个基于位置参数，当收到多个参数时，解析为元组。

与位置参数混合使用时候，位置参数必须在前。

可变数量关键字参数（收集为字典）

定义函数时使用 ** 开头的参数，可收集任意多个基于关键字形式的参数，当收到多个参数时，解析为字典

与位置参数混合使用时候，位置参数必须在前。

与可变数量位置参数混合使用时候，可变数量位置参数必须在前。

参数传递

位置传参

默认情况下，函数对接受到的参数进行 从左向右 依此按照位置匹配得到参数。

  • 代码示例

关键字传参

调用函数时，使用关键字参数， args=value ，明确指定传给函数的哪个参数，这种传递方式与顺序无关。

注意：传递方式为 参数名=值

  • 与位置参数混合使用时候

如果传参时混用位置参数和关键字参数，则关键字参数必须在位置参数之后。

序列解包传参（传递序列）

调用函数时候，向函数传递的参数为一个序列对象，并且在序列名称前加 * ，那么函数将分解序列，按位置分解得到参数。

  • 序列解包与位置参数混用
  • 定义时使用可变数量位置参数，调用时候使用序列解包

关键字解包传参（传递字典）

调用函数时候，向函数传递的参数为一个字典对象，并且在序列名称前加 ** ，那么函数将分解序列，按位置分解得到字典。

  • 定义参数时使用可变数量位置参数、可变关键字参数，调用时候使用序列解包

变量作用域

  1. 作用域 即变量的作用范围。
  2. 模块(module)，类(class)，函数(def、lambda) 会产生新的作用域。
  3. 条件判断(if……else……)、循环语句(for x in data)、异常捕捉(try……catch) 等不会产生新的作用域。

作用域划分

Python遵循LEGB原则：以 L → E → G → B 的规则查找，即从局部开始向上查找，找不到变量则报未定义错误。

  • B：build-in 系统固定模块里面的变量，也叫系统变量，比如int、open，解释器退出消亡。
  • G：global 全局变量，也就是模块级别定义的变量，模块导入时创建，线程退出消亡。
  • E：enclosing 嵌套的父级函数的局部作用域，就是包含此函数的上层函数的局部作用域
  • L：local 局部作用域，即为函数中定义的变量，函数调用时创建，调用结束消亡。

函数的变量在函数返回时，就会被销毁。

  • 函数内部可以访问全局变量
  • 在函数内部定义的变量，作用域在仅局部，即时与全局变量重名，由于作用域不同，全局也不会受到影响。
  • 在函数内部引用有重名的全局变量

改变作用域

  • 修改全局变量

若内部作用域想要修改外部作用域的变量，使用 global 关键字声明的变量，将作用于全局。

  • 修改嵌套作用域变量

如果需要修改嵌套作用域的变量，使用 nonlocal 关键字声明变量，将作用于嵌套层。

函数闭包

当调用一个函数时，默认情况下调用的函数一旦返回，则在其局部作用域的变量将被销毁，如果被调用的函数返回的是将内部子函数，而字函数又引用了外层函数的变量，那么此时子函数会自动记忆外层函数的变量，这种机制叫“函数闭包”

默认值参数作用域

函数的所有类型参数都是函数变量，属于局部变量，函数返回后就被销毁。

因为函数也是对象，Python把函数的默认值存放在属性中，而属性会伴随着函数的整个生命周期，不会因为调用结束而销毁，如果函数的默认值是一个可变的对象（如：列表），那么对这个列表的操作将被保留。

调用函数对象的 __defaults__ 能够显示属性，它保存了函数的所有参数默认值。

inspect

inspect模块主要提供了四种用处：

  1. 对是否是模块，框架，函数等进行类型检查。
  2. 获取源码
  3. 获取类或函数的参数的信息
  4. 解析堆栈

检查对象类型

  • is

用法：is{module|class|function|method|builtin}(object)

  • isroutine(obj)

用于检查对象是否为函数、方法、内建函数或方法等等可调用类型。用这个方法会比多个is()更方便，不过它的实现仍然是用了多个is()。如果目的是只要可以直接调用就需要是True的话，不妨使用isinstance(obj, collections.Callable)这种形式。

获取对象信息

  • getmembers(object[, predicate]):
    这个方法是dir()的扩展版，它会将dir()找到的名字对应的属性一并返回，形如[(name, value), …]。另外，predicate是一个方法的引用，如果指定，则应当接受value作为参数并返回一个布尔值，如果为False，相应的属性将不会返回。使用is*作为第二个参数可以过滤出指定类型的属性。
  • getmodule(object):
    还在为第2节中的module属性只返回字符串而遗憾吗？这个方法一定可以满足你，它返回object的定义所在的模块对象。
  • get{file|sourcefile}(object):
    获取object的定义所在的模块的文件名|源代码文件名（如果没有则返回None）。用于内建的对象（内建模块、类、函数、方法）上时会抛出TypeError异常。
  • get{source|sourcelines}(object):
    获取object的定义的源代码，以字符串|字符串列表返回。代码无法访问时会抛出IOError异常。只能用于module/class/function/method/code/frame/traceack对象。
  • getargspec(func): 仅用于方法，获取方法声明的参数，返回元组，分别是(普通参数名的列表, *参数名, **参数名, 默认值元组)。如果没有值，将是空列表和3个None。如果是2.6以上版本，将返回一个命名元组(Named Tuple)，即除了索引外还可以使用属性名访问元组中的元素。
  • getargvalues(frame): 仅用于栈帧，获取栈帧中保存的该次函数调用的参数值，返回元组，分别是(普通参数名的列表, *参数名, **参数名, 帧的locals())。如果是2.6以上版本，将返回一个命名元组(Named Tuple)，即除了索引外还可以使用属性名访问元组中的元素。
  • getcallargs(func[, *args][, kwds]):** 返回使用args和kwds调用该方法时各参数对应的值的字典。这个方法仅在2.7版本中才有。
  • getmro(cls): 返回一个类型元组，查找类属性时按照这个元组中的顺序。如果是新式类，与cls.mro结果一样。但旧式类没有mro这个属性，直接使用这个属性会报异常，所以这个方法还是有它的价值的。 返回当前的栈帧对象。

函数签名

  • signature(func)

可以得到函数的签名，是一个有序的字典

  • 得到函数返回值的类型注解
  • 得到函数签名中的参数
  • 得到参数的类型注解
  • 得到参数的默认值
  • 得到参数名称
  • 参数的传参方式

POSITIONAL_ONLY：值必须是位置参数提供
POSITIONAL_OR_KEYWORD：位置或关键字参数
VAR_POSITIONAL：可变数量位置参数，对应*args
KEYWORD_ONLY：关键字参数，对应或者args之后的出现的非可变关键字参数
VAR_KEYWORD：可变数量关键字参数，对应**kwargs

lambda 函数

lambda 是匿名函数：通俗地说就是没有名字的函数，lambda 函数没有名字，可以使用标识符来连接 lambda 函数。

lambda 函数有输入和输出：输入是传入到参数列表 args 的值，输出是根据表达式 expression 计算得到的值。

lambda 的首要用途是指定短小的回调函数。

lambda 是一个表达式而非语句，它不产生代码块。

语法

args: 以逗号分隔的参数列表
expression: 用到 args 的表达式(不能是语句)，不能是多条件的 if，可以是三元表达式，可以使用 for while 等。

  • 如果 source 为整数，则返回一个长度为 source 的初始化数组；

参数：iterable - 包含用于初始化frozenset的元素的迭代器，可以设置Iterable，字典，元组等。

queue队列

发表于 2017-09-28 | 分类于 Python

queue模块

FIFO队列Queue

创建队列

语法：queue.Queue(maxsize=0)

maxsize：队列的最大长度

写入数据.put

语法：put(self, item, block=True, timeout=None)

block：阻塞标志，表示当队列长度达到最大时是否阻塞。

timeout：阻塞时间，秒。

非阻塞写入.put_nowait

语法：put_nowait(self, item)

读出数据.get

阻塞读取并弹出数据

语法：get(self, block=True, timeout=None)

block：阻塞标志，表示是否阻塞。

timeout：阻塞时间，秒。

非阻塞读出.get_nowait

当弹出到最后一个元素时候，弹出异常。

队列长度.qsize

是否为空.empty

是否为满.full

阻塞调用.join

阻塞调用线程，直到队列中的所有任务被处理掉。

报告进度.task_done

主要是为 join配套使用，每次get后需要调用task_done，直到所有任务都task_done，join才取消阻塞。

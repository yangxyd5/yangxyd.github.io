random随机模块

发表于 2017-09-19 | 分类于 Python

random

Python提供的随机数生成模块

Function Description
randrange(start, stop[, step]) 返回范围内的随机整数
randint(a, b) 返回a和b之间的随机整数
choice(seq) 从非空序列中返回一个随机元素
shuffle(seq) 随机化对象的序列，就地修改
sample(population, k) 从可迭代对象中随机选择K个元素
random() 返回范围[0.0，1.0）中的下一个随机浮点数
uniform(a, b) 返回a和b之间的随机浮点数
seed(a=None, version=2) 初始化随机数发生器
getstate() 返回捕获生成器当前内部状态的对象
setstate(state) 恢复发生器的内部状态
getrandbits(k) 返回一个具有k个随机位的Python整数
triangular(low, high, mode) 返回低位和高位之间的随机浮点数，并在这些边界之间指定模式
betavariate(alpha, beta) Beta分布
expovariate(lambd)指数分布 指数分布
gammavariate(alpha, beta) 伽马分布
gauss(mu, sigma) 高斯分布
lognormvariate(mu, sigma) 记录正态分布
normalvariate(mu, sigma) 正态分布
vonmisesvariate(mu, kappa) Von Mises分配
paretovariate(alpha) 帕累托分布
weibullvariate(alpha, beta) 威布尔分布

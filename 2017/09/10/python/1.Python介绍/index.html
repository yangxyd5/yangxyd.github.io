Python介绍和环境

发表于 2017-09-10 | 分类于 Python

Python简介

Python 的设计具有很强的可读性，相比其他语言经常使用英文关键字，其他语言的一些标点符号，它具有比其他语言更有特色语法结构。

  • Python 是一种解释型语言：这意味着开发过程中没有了编译这个环节。类似于PHP和Perl语言。
  • Python 是交互式语言：这意味着，您可以在一个Python提示符，直接互动执行写你的程序。
  • Python 是面向对象语言: 这意味着Python支持面向对象的风格或代码封装在对象的编程技术。
  • Python 是动态、强类型语言：这意味着不用给变量指定数据类型，它会在第一次赋值给变量时在内部将数据类型记录下来。

垃圾回收

Python使用引用计数方式来进行垃圾回收的，当一个对象引用计数小于1的时候将被使用标记清除的方法从内存中清除，标记清除就是将对象标记为空白，这种算法的缺点是会产生内存碎片。

基础语法

  1. 转义序列：\,\t,\r,\n,\’,\”，具体见：man ascii，前缀r/R会让转义序列失效。
  2. 缩进：Python使用缩进的层级来表示代码层次关系。
  3. 代码续行：在行尾使用\，来表示一行未完成，下一行将继续，如果使用括号，认为括号内是一个整体。
  4. python 的变量名自定义函数名，只能包括字母数字下划线，且数字不能开头。
  5. 不能使用 python 内部的关键字，例如：def，class，等。

数据结构

  1. 列表：Python关键字标记为 list，用来表示“顺序的数据”
  2. 元祖：Python关键字标记为 tuple，用来表示“不可修改的顺序数据”
  3. 字典：Python关键字标记为 dict，用来表示“有别名的数据”
  4. 集合：Python关键字标记为 set，用来表示“没有顺序的数据”

非内置数据结构

  1. 堆：通常是由完全二叉树实现，可以看作一棵树的数组对象，堆是应用程序在运行的时候请求操作系统分配给自己内存，一般是申请/给予的过程，需要注意的是堆是在程序运行时，而不是在程序编译时。
  2. 栈：正好与队列相反，它实现了后进先出(LOFO)，对于堆栈的所有数据操作，都是在栈的顶部完成的，只可以查看栈顶部的数据，只能够向顶部压入数据，也只能在顶部弹出数据，这个结构主要用于递归。
  3. 队列：用来表示“先进先出”的数据(FIFO)，形象的比喻就像“毛毛虫”，它是先吃的东西，先消化拉出去。在开发中常用的消息处理，就是先来的消息，先“消化”，最后来的最后“消化”。
  4. 链表：链表是在非连续的内存单元中保存数据，并且通过指针将各内存单元相链接在一起，链表不需要事先申请固定大小的内存空间，对于大数据检索等比较常用。
  5. 二叉树：表示父子关系的数据，就更加抽象了，暂不了解。

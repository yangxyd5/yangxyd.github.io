列表解析式

列表解析是Python迭代机制的一种应用，它用于实现创建新的列表，因此列表解析的语句放在 [] 中，列表解析能够以比 for 快一倍的速度返回一个新的列表，同时语法简洁，在 Python 2.6 后加入。

语法

  1. [expression for iter_var in iterable]
  2. [expression for iter_var in iterable if cond_expression]
  3. [expression for iter_var in iterable iter_var in iterable]

其中：

  1. for 语句每次迭代的结果会被 expression 处理而产生一个新的元素
  2. for 语句中 if 子句，则表示只有符合条件的才会被 expression 处理成为新元素
  3. for 语句可以嵌套执行，从而实现两个列表交叉

示例

  • 基于已有列表创建新列表，新列表的元素为已有列表元素的平方
  • 基于已有列表创建新列表，新列表的元素为前一个元素加后一个元素的和
  • 生成 '0001.ntumkbwdvj' - '0100.yerxcbgntw' 这种形式的新列表，其中后半部分为随机字母。

  • 打印九九乘法表

  • 列出以 .pdf 结尾的所有文件

  • 生成类似多项式相乘的元组列表

生成器写法

  • 将 [ ] 改为 ()，可以实现惰性求值

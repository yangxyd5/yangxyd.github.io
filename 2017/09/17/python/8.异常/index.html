try语法

try过程

  • try 会开启一个上下文，然后执行 try 与 except 之间的语句，语句将决定控制流能否正常通过try
  • 如果 try 与 except 之间的语句发生异常，则立即从异常处跳出到 try 上下文，try 语句捕获异常后，将异常交给第一个匹配的 except 子句，执行except子句的代码，完毕后返回 try ，然后控制流正常通过 try 语句
  • 如果 try 与 except 之间的语句发生异常，则立即从异常处跳出到 try 上下文，try 语句捕获异常后，没有找到匹配的 except 子句，则 try 会将异常向上层抛出，并且不再向后执行同级代码。
  • 如果 try 与 except 之间的语句没有发生异常，则 try 语句会执行 else 子句的代码，然后控制流正常通过 try 语句
  • 无论程序控制流是否正常通过try语句，都会在最后执行finally 子句

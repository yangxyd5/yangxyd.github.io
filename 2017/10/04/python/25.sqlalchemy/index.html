方法 作用
.filter_by(**kwargs) 根据条件过滤，如 session.query(MyClass).filter_by(name = 'some name', id = 5)
.filter(*criterion) 根据条件过滤，比.filter_by()灵活，支持表达式，如 session.query(MyClass).filter(MyClass.name == 'some name', MyClass.id > 5)
.order_by(*criterion) 与 SQL 的 ORDER BY 类似，默认升序，需要时可以 sqlalchemy.desc() 降序
.limit(limit) 与 SQL 的 LIMIT 类似
.offset(offset) 与 SQL 的 OFFSET 类似
.all() 返回匹配查询的所有对象
.one() 返回匹配查询的唯一一个（下一个）对象
.first() 返回匹配查询的第一个对象
join(*props, **kwargs) 按给定的 JOIN 条件创建 SQL JOIN 语句
update(values, synchronize_session='evaluate', update_args=None) 批量更新行
delete(synchronize_session='evaluate') 批量删除行

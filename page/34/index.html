正则表达式

元字符 含义
. 匹配除换行符之外的任何字符
^ 匹配字符串的开头
$ 匹配字符串的结尾或紧挨着换行符之前字符串的结尾
* 匹配前面表达式的0个或更多（贪婪）重复，贪婪意味着它会匹配尽可能多的重复
+ 匹配前面表达式的1个或更多（贪婪）重复
? 匹配上述表达式的0或1（贪婪）
*?、+?、?? 分别表示上面三个特殊字符的非贪婪版本
{m,n} 匹配前面表达式的m到n次重复
{m,n}? 上一个的非贪婪版本
[] 字符的集合，匹配该集合内任意单个字符，当第一个字符为 ^ 时，则表示匹配不属于该集合的任意单个字符
\ 表示或关系，A表达式 或 B表达式
(…) 匹配并获取为分组，储存该子字符串用于使用分组号向后引用它
(?:…) 匹配但不获取分组，不存储匹配的子字符串用于向后引用
(?P\<name>…) 匹配并获取为命名的分组，使用名称引用该分组
(?P=name) 匹配名为name的组之前匹配的文本
(?=…) 正向匹配预查，如果前面表达式的结果的后面字符串，被此表达式匹配，前面表达式才会成立，但只输出前面表达式的结果
(?!…) 正向不匹配预查，如果前面表达式的结果的后面字符串，不被此表达式匹配，前面表达式才会成立，但只输出前面表达式的结果
(?<=…) 反向匹配预查，如果前面表达式的结果的前面字符串，被此表达式匹配，前面表达式才会成立，但只输出前面表达式的结果
(?<!…) 反向不匹配预查，如果前面表达式的结果的前面字符串，不被此表达式匹配，前面表达式才会成立，但只输出前面表达式的结果
\number 匹配相同编号的分组内容，向后引用
\A 仅匹配字符串的开头
\Z 仅匹配字符串的末尾
\b 匹配一个单词边界，也就是指单词和空格间的位置
\B 匹配非单词边界，也就是该字符不能是单词的边界
\d 匹配一个数字字符。等价于[0-9]
\D 匹配一个非数字字符。等价于[^0-9]
\s 匹配任何空白字符，包括空格、制表符、换页符等等。等价于[ \f\n\r\t\v]
\S 匹配任何非空白字符，等价于[^ \f\n\r\t\v]
\w 匹配任何字母数字字符，等价于[a-zA-Z0-9_]
\W 匹配任何非单词字符。等价于[^a-zA-Z0-9_]
\xNumber 匹配Number，Number为十六进制转义值的ASCII编码字符，例如，\x41匹配A
\uNumber 匹配Number，Number为4个十六进制数字表示的Unicode字符，例如，\u00A9匹配©符号

re模块

re 模块的一般使用步骤如下：

  • 使用 compile 函数将正则表达式的字符串形式编译为一个 Pattern 对象
  • 通过 Pattern 对象提供的一系列方法对文本进行匹配查找，获得匹配结果（一个 Match 对象）
  • 最后使用 Match 对象提供的属性和方法获得信息，根据需要进行其他的操作

预编译.compile

编译一个正则表达式模式，返回一个模式对象，此处应该使用原始字符串 r 。

语法：compile(pattern, flags=0)

flag： 是一个可选参数，表示匹配模式，比如忽略大小写，多行模式等。

开头匹配.match

语法：match(pattern, string, flags=0)

pattern：

完全匹配.fullmatch

语法：fullmatch(pattern, string, flags=0)

搜索匹配.search

搜索匹配，找到一个结果立即返回一个位置，不再继续匹配。

语法：search(pattern, string, flags=0)

替换字串.sub

使用repl替换string中每一个匹配的子串后返回替换后的字符串。

语法：sub(pattern, repl, string, count=0, flags=0)

count：用于指定最多替换次数，不指定时全部替换。

替换字串返回次数.subn

使用repl替换string中每一个匹配的子串后返回替换后的字符串和替换次数。

语法：subn(pattern, repl, string, count=0, flags=0)

分割为列表.split

语法：split(pattern, string, maxsplit=0, flags=0)

全部结果.findall

搜索string，以列表形式返回全部能匹配的子串。

语法：findall(pattern, string, flags=0)

全部结果迭代器.finditer

语法：finditer(pattern, string, flags=0)

序列化

  • 序列化

将一段数据、对象，转换为字节流，以便保存到文件、数据库或经过网络传输的过程称之为序列化。

  • 反序列化

在需要时候重新打开已序列化数据，或者在另外的计算机上需要恢复，通过相同序列化格式恢复得到原来的数据结构，这个过程称作反序列化。

用法：loads(data, *, fix_imports=True, encoding='ASCII', errors='strict')

用法：dump(obj, file, protocol=None, *, fix_imports=True)

从文件还原.load

用法：load(file, *, fix_imports=True, encoding='ASCII', errors='strict')

csv.reader()

  • 创建对象

语法：reader(iterable [, dialect='excel'] [参数=值])

iterable ：必须是一个可迭代的对象，例如打开的文件，有 __next__()方法的。

dialect ：解析规则，默认是用逗号分隔的 excel ，其他可用值：['excel', 'excel-tab', 'unix']

参数 含义
delimiter = ‘,’ 用作字段分隔的字符
quotechar = ‘“‘ 用作引用的字符
escapechar = None 用作转义的字符
doublequote = True 连续两次引号是否被解释为一个引号
skipinitialspace = False 是否忽略分隔符后面跟着的空格
lineterminator = ‘\r\n 行结束符
quoting = QUOTE_MINIMAL 写入时候如何生成引号，默认仅在需要时生成，QUOTE_ALL表示总是生成
  • csv.reader支持的方法

__next__()：得到csv对象的下一行内容列表

line_num：从csv对象读取的行数

fieldnames ：取得字段名称

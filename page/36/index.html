文件IO

文件的操作

创建文件.open()

打开文件和创建文件对象，文件属于可迭代对象。

语法

open(file, mode='r', buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)

参数

  • file 文件路径或者打开文件的描述符，必须的参数。
  • mode 文件的打开模式
模式 作用
r 读模式（默认）
w 写模式，从首部直接覆盖文件
x 创建并写入一个新文件
a 追加模式
b 二进制模式，配合 rw 等参数使用
t 文本模式（默认）
+ 读写模式，补全缺失的模式
  • buffering 设置缓冲的策略，正数表示使用指定大小的缓冲
参数 含义
0 表示禁用缓冲
负数 表示使用系统默认缓冲
1 表示缓冲一行数据
大于1 表示指定大小的缓冲。
  • encoding 文本文件的编码设置，windows 默认：GBK，LINUX默认：UTF-8。
  • errors 指定编码错误的可选字符串。
  • newline 控制通用换行符的工作方式。

在读文件的时候：newline=None 表示\r、\n、\r\n都会被转换为\n

在写文件的时候：newline=None 表示使用系统的分行符号。

示例

打开模式

模式 描述
r 以只读方式打开文件。文件的指针将会放在文件的开头。这是默认模式。
rb 以二进制格式打开一个文件用于只读。文件指针将会放在文件的开头。这是默认模式。一般用于非文本文件如图片等。
r+ 打开一个文件用于读写。文件指针将会放在文件的开头。
rb+ 以二进制格式打开一个文件用于读写。文件指针将会放在文件的开头。一般用于非文本文件如图片等。
w 打开一个文件只用于写入。如果该文件已存在则将其覆盖。如果该文件不存在，创建新文件。
wb 以二进制格式打开一个文件只用于写入。如果该文件已存在则将其覆盖。如果该文件不存在，创建新文件。一般用于非文本文件如图片等。
w+ 打开一个文件用于读写。如果该文件已存在则将其覆盖。如果该文件不存在，创建新文件。
wb+ 以二进制格式打开一个文件用于读写。如果该文件已存在则将其覆盖。如果该文件不存在，创建新文件。一般用于非文本文件如图片等。
a 打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。
ab 以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。
a+ 打开一个文件用于读写。如果该文件已存在，文件指针将会放在文件的结尾。文件打开时会是追加模式。如果该文件不存在，创建新文件用于读写。
ab+ 以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。如果该文件不存在，创建新文件用于读写。

关闭文件.close()

调用 close 方法以后文件对象还存在，但是不能再进行操作了，每次不需要对文件操作的时候需要显式执行关闭操作。

读取内容.read()

从文件中读取指定个字符，指针会移动到读取后的位置，默认读取到文件尾部。

参数：read(size=-1)

读取一行.readline()

生成每行列表.readlines()

移动指针.seek()

把当前的操作指针移动从whence 定义的位置开始偏移 offset 的字节量。

  • 用法：seek(offset [,whence])

offset 偏移量，（当 whence 为1时可以为负数）

whence 指定偏移的开始位置

参数 含义
0 whence的默认值，表示从文件的头部开始
1 表示从当前位置开始
2 表示从文件的尾部开始

上下文管理

文件对象打开的时候会获得一个系统的文件描述符，计算机的文件描述符资源是有限的，为了保护资源不被耗尽，所以可以使用with语句来自动关闭文件对象。

上下文管理的语句块不会开启新的作用域，with语句块执行完的时候，会自动关闭文件对象。

.datetime() 日期对象

datetime对象可以用来表示精确的日期和时间，其实例化方法如下

.strftime() 格式化为字符串

返回自定义格式化字符串表示日期

语法：date.strftime(format)

.strptime() 字符串转时间

将字符串格式化为时间对象

语法：datetime.strptime(date_string, format)

  • 格式化符号
指令 含义
%a 星期的简写（星期一 ~ 天：Mon, Tue, Wed, Thu, Fri, Sat, Sun）
%A 星期的全写（星期一 ~ 天：Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday）
%w 在一个星期中的第几天（ 0 表示星期天 … 6 表示星期六）
%d 在一个月中的第几天（01, 02, …, 31）
%b 月份的简写（一月 ~ 十二月：Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec）
%B 月份的全写（一月 ~ 十二月：January, February, March, April, May, June, July, August, September, October, November, December）
%m 月份（01, 02, …, 12）
%y 用两个数字表示年份（例如 2014年 == 14）
%Y 用四个数字表示年份
%H 二十四小时制（00, 01, …, 23）
%I 十二小时制（01, 02, …, 11）
%p AM 或者 PM
%M 分钟（00, 01, …, 59）
%S 秒（00, 01, …, 59）
%f 微秒（000000, 000001, …, 999999）
%z 与 UTC 时间的间隔 ；如果是本地时间，返回空字符串（(empty), +0000, -0400, +1030）
%Z 时区名称；如果是本地时间，返回空字符串（(empty), UTC, EST, CST）
%j 在一年中的第几天（001, 002, …, 366）
%U 在一年中的第几周，星期天作为第一天（00, 01, …, 53）
%W 在一年中的第几周，星期一作为第一天（00, 01, …, 53）
%c 用字符串表示日期和时间（Tue Aug 16 21:30:00 2014）
%x 用字符串表示日期（08/16/14）
%X 用字符串表示时间（21:30:00）
%% 表示百分号

Golang介绍

语法规范

  1. Go 源码是以 .go 为扩展名的
  2. Go 应用程序的执行入口是 main() 函数
  3. 严格区分大小写
  4. Go 代码的每个语句结尾不需要加 ;
  5. Go 编译器是一行一行编译的，因此一行就写一条语句，否则报错
  6. Go 语言 import 包如果没有使用到，那么代码不能编译通过

关于 GC

  • GC 英文名称全程：gtbage collector，一些常用的 GC 算法：
算法 描述
引用计算法 当对象被引用时计数器增加一，不被引用时计数器减少一，PHP 和 Object-C 使用，缺点是：互相引用的无法回收，计数器增加消耗。
标记清除 停止程序运行，遍历对象，进行标记，标记完成后将没有引用的对象进行清除，缺点是：由于标记清除需要停止程序，当对象特别多的时候，标记和清除的过程比较耗时
三色标记法 是标记清除的改进版，从逻辑上分为白色区域（未搜索）、灰色区域（正在搜索）、黑色区域（已经搜索）灰色区域内容是子引用没有进行搜索，黑色区域表示子引用已经存在
分代收集 一般有三代，例如：Java 新生代、老年代、永久代，当新生代中达到阈值时会把对象放入老年代，相同道理当老年代内容达到阈值，会放入永久代

变量

Go 语言的变量是有明确类型的，变量名称应该遵循驼峰法则，Go 语言中一个变量被声明之后，系统自动赋予该类型的零值，int 为 0，float 为 0.0 ，bool 为 false 等，指针为 nil。

变量声明

常量

常量是一个固定值，在编译期就确定结果，声明时候必须赋值且结果不可以改变，因为常量在编译期就确定可以方式程序运行过程中意外修改。

常量使用关键字 const ，常量定义完以后可以不使用；

常量可以是表达式，但不允许出现变量；

常量声明时候可以不指定类型，使用默认类型；

没有指定类型的常量，可以直接进行运算(不进行类型转换)；

变量地址

变量本质上就是内存中一块数据的标记，把值存储到变量中实质上是把值存储在内存中，每次对变量重新赋值就是在修改变量地址中的内容。

获取变量内存地址

创建基本类型变量会开辟新地址

  • 新创建的基本类型变量会开辟新的内存地址
  • 对基本类型变量的修改只影响自己

  1. 有符号整形：它的二进制最高位表示符号，0 为正数、1 为负数，int 和 intx 为有符号整形

切片

切片的英文名称 slice，切片是具有可变长度，相同类型元素的序列，由于长度是可变的，可以解决数组长度在数据不确定的情况下浪费内存的问题。

  • 声明语法，和数组的唯一区别是不需要声明长度

make初始化切片

在 go 语言中，可以使用 make 函数创建 slice、map、channel、interface

切片的容量与长度

长度是切片元素个数，容量切片可以有的元素个数。

长度可可以使用索引访问，超出索引的容量还没有元素，所以不能使用索引访问。

  • 使用 make 函数定义的切片没有内容，但这个切片已经申请了内存空间。

超出容量则翻倍

  • 当 append 超出了切片的容量，那么切片的容量会翻倍增加

通过数组产生切片

  • 取出数组的一部分就会产生一个切片
  • 切片是指针，指向数组元素地址，修改切片的内容，数组的内容会发生变化
  • 当向数组产生的切片添加内容时，切片增加的部分实际是在修改原数组的内容
  • 当切片超出数组长度时候，超出部分会被分配新的内存空间，同时内存地址发生变动

切片本质上就是一个指针

比较运算符

运算符 含义
== 检查两个操作数的值是否相等，如果相等，则条件为真。
!= 检查两个操作数的值是否相等，如果值不相等，则条件为真。
> 检查左操作数的值是否大于右操作数的值，如果是，则条件为真。
< 检查左操作数的值是否小于右操作数的值，如果是，则条件为真。
>= 检查左操作数的值是否大于或等于右操作数的值，如果是，则条件为真。
<= 检查左操作数的值是否小于或等于右操作数的值，如果是，则条件为真。

逻辑运算符

运算符 含义
&& 逻辑AND运算符。如果两个操作数都不为零，则条件为真。
\ \ 逻辑OR运算符。如果两个操作数中的任何一个非零，则条件变为真。
! 逻辑非运算符。用于反转其操作数的逻辑状态。如果条件为真，则逻辑非运算符将为假。

位操作

运算符 含义
& 如果两个操作数中都存在二进制AND运算符，则将其复制到结果。

^ 二进制XOR运算符复制位，如果它在一个操作数中设置，但不是在两个操作数中设置。
<< 二进制左移位运算符。左操作数值向左移动由右操作数指定的位数。
>> 二进制右移运算符。左操作数值向右移动由右操作数指定位数。

指针

指针变量指向值的内存地址，指针的主要应用是实现在多个地方操作同一个内存地址的值。

概念 说明
变量 是一种占位符，用于引用计算机的内存地址。可理解为内存地址的标签
指针 表示内存地址，表示地址的指向。指针是一个指向另一个变量内存地址的值
& 取地址符，例如：{指针}:=&{变量}
* 取值符，例如：{变量}:=*{指针}
